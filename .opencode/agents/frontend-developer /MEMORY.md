# Project Memory: Tech E-Commerce Frontend

## Tech Stack

- Framework: Next.js (App Router)
- Package Manager: Bun
- Styling: Tailwind CSS
- State: Zustand (Persisted)
- Animations: Framer Motion
- Icons: Lucide-React
- Validation: Zod

## Core Views

1. Home (`/`) - Hero Carousel, Promo Banners, About block.
2. Products Grid (`/products`) - Category filtering.
3. Product Details (`/products/[category]/[brand]/[model]`) - High-quality images, specs, cart actions.
4. Profile (`/profile`) - Avatar cycle, name update, delete account.
